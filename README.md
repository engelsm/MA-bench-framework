# HPC Memory Encryption Benchmark Suite

This repository contains a benchmark framework for evaluating the performance impact of a subclass of HPC workloads, with a focus on sparse linear algebra.

The benchmarks were used for analyzing hardware-based memory encryption mechanisms, specifically AMD SME and SEV, and their impact on HPC performance.

The repository includes:
- benchmark source code
- benchmark execution scripts
- collected results
- analysis notebooks
- supporting artifacts

## Benchmark Overview

The suite consists of three main workload categories:

- **STREAM**: Measures sustainable memory bandwidth  
- **SpMV (Sparse Matrix-Vector Multiplication)**: Represents memory-bound sparse linear algebra kernels  
- **Krylov Solvers**: Higher-level iterative methods built on top of SpMV  

## Experimental Design

The framework supports systematic parameter sweeps for SpMV and Krylov solvers across:

- thread counts  
- matrix sizes and sparsity patterns  
- process-level NUMA policies  

For the thesis evaluation, benchmarks were executed on virtual machines using the following execution modes:

- Native (unencrypted baseline)  
- SME  
- SME + SEV  

Additionally, NUMA memory placement policies were varied at the VM configuration level.
